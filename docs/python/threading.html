Threading

disable_threading

dcgpy.disable_threading()

Disables threading on a global level.

Example:

>>> from dcgpy import *
>>> disable_threading()

enable_threading

dcgpy.enable_threading()

Enables threading on a global level (if disabled by calling dcgpy.disable_threading()).

Example:

>>> from dcgpy import *
>>> disable_threading()
>>> enable_threading()
